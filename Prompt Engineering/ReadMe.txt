# Prompt Engineering Log Analyzer

This project demonstrates prompt engineering using a Large Language Model (Gemini).

## Features
- Classifies logs as Normal, Suspicious, or Malicious
- Uses few-shot prompting
- No model training required

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Add your API key in app.py

3. Run:
   python app.py
